Программа 'goldbach' проверяет гипотезу бинарной проблемы Гольдбаха на заданных значениях
Для сборки и компиляции проекта используйте команду make
Для запуска программы используйте ./goldbach
make clean - удаление объектных файлов
make fullclean - удаление объектных и исполняемых файлов
make arch - создание дистрибутива
make manual - инструкция
make help - помощь по функциям
make wc - подсчет строк исходного кода проекта
make indent - задает отступы и выравнивание во всех файлах с исходным кодом
make copy - копирование исполняемого файла в корневую директорию
